# mkdocs-doubleslash-theme

A MkDocs plugin that applies [doubleSlash style](https://doubleslash.style/) to your documentation.

## Installation

```shell
pip install mkdocs-doubleslash-theme
```

Then add the plugin to your `mkdocs.yml`:

```yaml
theme: material

plugins:
  - search
  - mkdocs-doubleslash-theme
```

## Documentation

For more information, see the [full documentation](https://doubleslashde.github.io/mkdocs-doubleslash-theme).
